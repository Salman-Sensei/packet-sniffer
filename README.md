# Packet Sniffer

A lightweight packet sniffer project built using Python and raw sockets.

## Goals
- Capture live network packets
- Analyze TCP, UDP, and ICMP traffic
- Build a Wireshark-like interface using PyQt5
- Learn low-level networking and cybersecurity concepts

## Planned Features
- Live packet capturing
- Protocol analysis
- Packet filtering
- GUI dashboard
- Traffic statistics
- Export captured packets

## Tech Stack
- Python
- Socket Programming
- PyQt5
- Linux Raw Sockets
